# 提交本地代码到新分支

将从分支 A 下载代码，然后提交到分支 B 添加本地需要提交代码：

git add .

// 提交本地代码：
git commit -m "添加到分支B"

// push 到 git 仓库：
git push origin branchA:branchB

// 切换新分支：
git checkout -b branchB origin/branchB
